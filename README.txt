The Plugin creates a vacancy calendar for holiday homes.

== Description ==

Holiday Homes Vacancy is a plugin, that creates a calendar by including a shortcode. Occupied Days can be entered and will be highlighted in the calendar.


*   Enter the occupied days (in the navigation under "Occupied")
*   Add the shortcode to see the calendar
*   The Language settings and the name of the shortcode can be found in the navigation under "Occupied" -> "Settings"


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `holiday homes vacancy` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter the occupied days under "Occupied" in the navigation
